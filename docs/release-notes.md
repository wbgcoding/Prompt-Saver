## Prompt Saver 1.7.0

### Downloads

| File | Use it when |
|---|---|
| `Prompt.Saver_1.7.0_x64-setup.exe` | Installer — choose **all users** (admin) or **current user only**; upgrades replace the previous version automatically, your data is kept |
| `prompt-saver.exe` | Portable standalone version — one file, no installation |

Requires the Microsoft WebView2 runtime (preinstalled on Windows 11 and current Windows 10). If it is missing, the app offers the official Microsoft installer on first start.

### Added
- **Collapsible bars**: subtle arrows hide the top bar and the input bar — the grid grows to use the freed space, the choice survives restarts, and small floating arrows bring each bar back
- **Grid-size picker**: the size fields open a scrollable dropdown with all values; the current one is highlighted and the mouse wheel steps through

### Improved
- Settings: font and text size now have their own labelled dropdowns
- The settings scrollbar stays inside the rounded corners
- The image and hide buttons in the input bar line up with the Save button

## Prompt Saver 1.6.2

### Downloads

| File | Use it when |
|---|---|
| `Prompt.Saver_1.6.2_x64-setup.exe` | Installer — choose **all users** (admin) or **current user only**, with desktop / start menu shortcuts and "run after install" (all pre-selected, all optional) |
| `prompt-saver.exe` | Portable standalone version — one file, no installation |

Requires the Microsoft WebView2 runtime (preinstalled on Windows 11 and current Windows 10). If it is missing, the app offers the official Microsoft installer on first start.

### Improved
- Upgrading over an existing installation no longer asks about the previous version — it is removed automatically (your prompts and settings are kept)

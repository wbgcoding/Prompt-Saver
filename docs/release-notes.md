## Prompt Saver 1.5.0

### Downloads

| File | Use it when |
|---|---|
| `Prompt.Saver_1.5.0_x64-setup.exe` | Installer — choose **all users** (admin) or **current user only**, with desktop / start menu shortcuts and "run after install" (all pre-selected, all optional) |
| `prompt-saver.exe` | Portable standalone version — one file, no installation |

Requires the Microsoft WebView2 runtime (preinstalled on Windows 11 and current Windows 10). If it is missing, the app offers the official Microsoft installer on first start.

### Added
- **Automatic updates**: the app checks GitHub once a day and notifies you with an **Install now** button; manual check & install under Settings → Updates
- Current version shown in the settings

### Fixed
- Tile text can no longer overflow its button when resizing the window right after startup

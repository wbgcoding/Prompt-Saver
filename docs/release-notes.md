## Prompt Saver 1.3.3 — HotFix

### Downloads

| File | Use it when |
|---|---|
| `Prompt.Saver_1.3.3_x64-setup.exe` | You want a normal installation — choose **all users** (admin) or **current user only**, with desktop / start menu shortcuts and "run after install" (all pre-selected, all optional) |
| `prompt-saver.exe` | You want the portable standalone version — one file, no installation |

Requires the Microsoft WebView2 runtime (preinstalled on Windows 11 and current Windows 10). If it is missing, the app offers the official Microsoft installer on first start.

### Fixed
- Tile text no longer shifts while dragging a prompt
- Letters like **g, j, p, y** are no longer cut off at the bottom of buttons

### Improved
- Installer: new choice between **install for all users** and **only for me**

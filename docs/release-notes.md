## Prompt Saver 1.6.0

### Downloads

| File | Use it when |
|---|---|
| `Prompt.Saver_1.6.0_x64-setup.exe` | Installer — choose **all users** (admin) or **current user only**, with desktop / start menu shortcuts and "run after install" (all pre-selected, all optional) |
| `prompt-saver.exe` | Portable standalone version — one file, no installation |

Requires the Microsoft WebView2 runtime (preinstalled on Windows 11 and current Windows 10). If it is missing, the app offers the official Microsoft installer on first start.

### Added
- **Auto update toggle** in the settings (on by default), with a tooltip explaining the once-a-day check

### Improved
- Updates install **fully automatically**: silent installer, no clicks, the app restarts itself on the new version

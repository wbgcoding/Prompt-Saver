## Prompt Saver 1.3.2

### Downloads

| File | Use it when |
|---|---|
| `Prompt.Saver_1.3.2_x64-setup.exe` | You want a normal installation (current user, no admin needed) with desktop / start menu shortcuts and "run after install" — all pre-checked and optional |
| `prompt-saver.exe` | You want the portable standalone version — one file, no installation |

Requires the Microsoft WebView2 runtime (preinstalled on Windows 11 and current Windows 10). If it is missing, the app offers the official Microsoft installer on first start.

### Highlights

- **Floating buttons overhauled**: fully transparent background, right-click menu (size S/M/L, edit prompt, remove), text matches the grid tiles, spawn at the top-left of the primary monitor — and the freeze when toggling them is fixed for good
- **Auto-fit tile text**: fills the whole button with clean word wrapping, never clipped
- **Localized start page**: the default view name follows the UI language (10 languages)
- **NSIS installer** alongside the portable exe (this release)
- New defaults: 5×4 grid, minimize-to-tray off
- Cleanups: Windows-only build (no mobile leftovers), smaller icon set, leaner repo
